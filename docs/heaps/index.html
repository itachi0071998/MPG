Priority Queue#

To save you the time of implementing heaps and prevent you from making unnecessary mistakes, STL in C++ provides Priority Queue which is nothing but by default max heap. Keep on reading and see for yourself what priority queue is all about

important

The following STL utilities for Priority Queues in C++ are a must know : empty(), size(), push(),pop(),top().
